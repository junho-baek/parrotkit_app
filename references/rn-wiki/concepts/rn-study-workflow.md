# RN 학습 워크플로우 | RN Study Workflow

## 요약

이 워크스페이스는 계획 우선 방식으로 React Native를 공부한다. 미래에 볼 소스는 먼저 올려둘 수 있지만, 실제 ingest는 사용자가 특정 챕터, 기능, 코드 조각을 시작했을 때만 깊게 진행한다.

## 현재 이해

- 레거시 소스 저장소는 `raw/` 아래에 불변 참고 자료로 둔다.
- 노마드 코더 React Native 마스터클래스와 `raw/noovies/`를 현재 주된 staged source로 취급한다.
- 소스 수준 결론을 적기 전에 먼저 학습 의도와 범위를 기록한다.
- 실제 학습이 시작되면 한 번에 작은 범위 하나만 ingest 한다.
- 예전 강의 패턴과 최신 React Native 가이드는 자동 개편이 아니라 비교 대상으로 다룬다.
- 최신 비교 기준은 [`vercel-react-native-skills`](/Users/junho/.agents/skills/vercel-react-native-skills/SKILL.md)로 두고, `oh-my-rn`은 personal overlay로만 다룬다.
- 위키에 남긴 구현 가능한 결론은 `nomad-diary/`에도 같은 흐름 안에서 계속 반영한다.
- 바로 구현하지 못한 결론은 왜 보류했는지와 다음 액션을 위키에 같이 남긴다.
- `wiki/analyses/`의 제목과 파일명은 커밋 번호보다 오래 남는 개념을 먼저 드러내고, 커밋/챕터 정보는 `범위` 같은 메타 섹션으로 내린다.
- 반복되는 교훈은 재사용 가능한 개인용 스킬 `oh-my-rn`으로 승격시킨다.
