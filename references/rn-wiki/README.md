# RN 위키 스냅샷 | RN Wiki Snapshot

이 폴더는 `/Users/junho/project/RN_practice/wiki`에서 가져온 React Native 학습 위키의 참조용 스냅샷이다.

## Entry Points

- [index](index.md): 위키 전체 목차
- [overview](overview.md): 현재 학습 범위와 상태 요약
- [log](log.md): 시간순 작업 기록

## Included

- `analyses/`: 기능, 패턴, 비교 메모
- `concepts/`: 장기적으로 유지할 개념 문서
- `sources/`: 실제 학습이 시작된 소스 요약

## Excluded

- 원본 `raw/` 소스 디렉터리
- Obsidian 개인 작업 상태 파일(`.obsidian/workspace.json` 등)

## Portability Notes

- 일부 문서 링크는 원래 학습 워크스페이스나 개인 스킬 경로(`/Users/junho/...`)를 그대로 가리킨다.
- 이 스냅샷은 우선 참조성과 보존성을 위해 복사했으며, 경로 정규화는 필요할 때 별도로 진행한다.
